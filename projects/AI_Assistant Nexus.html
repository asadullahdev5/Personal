AI-Assistant-Nexus

  • Tech Stack: Python (pyttsx3, speech_recognition, pywhatkit)
  • Github URL: Project Link

Key Features:

  • Voice Activation: Nexus is always on standby, ready to activate with the command "wake up."
  • Time Teller: Quickly provides the current time upon request.
  • Web Automation: Automates browsing tasks by opening LinkedIn, GitHub profiles, or performing searches on the Brave browser.
  • Wikipedia Integration: Fetches concise summaries from Wikipedia based on user queries.
  • Music Playback: Plays requested songs directly from YouTube, offering an easy way to enjoy music.
  • Smart Typing: Converts spoken words into text in a notepad, facilitating quick note-taking.

Technical Implementation:

  • Utilizes pyttsx3 for text-to-speech functionality, enabling Nexus to communicate audibly with the user.
  • Employs speech_recognition to capture and process voice commands accurately.
  • Leverages pywhatkit for automating web-related tasks, enhancing the assistant's efficiency.

Performance Testing:

Nexus has been tested for its responsiveness and accuracy, ensuring it delivers fast and reliable assistance.

Future Plans:

The development of Nexus is ongoing, with plans to integrate additional features that further enhance its capabilities.
